FinOps Tool — Session Summary
Date: 2026-03-10
Model: Claude Sonnet 4.6

================================================================================
OBJECTIVE
================================================================================
Build a simple, portable FinOps tool in Python that:
  - Logs into a cloud account (Azure first, modular for future providers)
  - Extracts cloud resource usage across multiple subscriptions
  - Maps usage to current retail prices via public APIs
  - Exports results to CSV
  - Generates an Excel dashboard for visualisation
  - Can also work with a pre-existing CSV file as input

================================================================================
DECISIONS & DESIGN CHOICES
================================================================================
  - Language:       Python 3.9+
  - Auth:           Azure CLI (az login) via DefaultAzureCredential
  - Pricing:        Azure Retail Prices REST API (free, no auth required)
  - Modularity:     Abstract BaseProvider class — AWS/GCP placeholders ready
  - Missing data:   Text fields use "NDF" (No Data Found); numeric fields use 0
  - Terminal output: Colour-coded via colorama (cyan=headers, green=success,
                     yellow=warnings, red=errors, magenta=cost totals)
  - Storage pricing: 100 GB placeholder quantity (actual GB unknown without
                     consumption API)

================================================================================
SUPPORTED RESOURCE TYPES (initial list in resources.yaml)
================================================================================
  - Microsoft.Compute/virtualMachines
  - Microsoft.Storage/storageAccounts
  - Microsoft.Sql/servers/databases      (nested type — handled separately)
  - Microsoft.Web/serverFarms
  - Microsoft.Network/virtualNetworks    (no direct cost — returns $0)

================================================================================
CSV OUTPUT COLUMNS
================================================================================
  subscription_id | subscription_name | resource_group | resource_name |
  resource_type | location | sku | size | unit | quantity |
  unit_price_usd | estimated_cost_usd

================================================================================
CLI FLAGS
================================================================================
  --provider azure              Cloud provider (required)
  --output FILE                 Output CSV path (required)
  --input FILE                  Load existing CSV instead of live extraction
  --subscription ID [ID ...]    Target specific subscription(s)
  --config FILE                 Custom resources.yaml path
  --no-pricing                  Skip pricing API — inventory only
  --dashboard FILE              Also generate an Excel dashboard (.xlsx)

================================================================================
EXCEL DASHBOARD SHEETS
================================================================================
  1. Summary      — KPI cards, cost by resource type (table + bar chart),
                    cost by subscription (table + pie chart)
  2. By Location  — Regional breakdown table + horizontal bar chart
  3. Raw Data     — Full filterable Excel Table, frozen header,
                    NDF cells highlighted yellow, zero-cost rows in grey
